# PetroNet-Real-Time-Status-Panel


This repo contains a production-oriented starter for a real-time industrial monitoring dashboard.


- Frontend: React + Vite (served via Cloudflare Pages in production)
- Backend: Firebase Realtime Database + Cloud Functions (alerts)
- Maps: Leaflet + OpenStreetMap tiles
- Local development: docker-compose optional, sensor simulator included


See `frontend/README.md` and `functions/README.md` for setup.
